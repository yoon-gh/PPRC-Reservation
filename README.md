# 표현체 연구시설 예약 시스템

React + Vite + Supabase 기반 프로토타입입니다.

##
URL : https://pprc-reservation.vercel.app/

## 기능

- 사용자 예약 신청
- Supabase 공용 DB 저장
- 새로고침/다른 PC에서도 예약 공유
- 관리자 이메일 로그인
- 관리자 승인/반려/수정/삭제
- 현재 캘린더 월 기준 CSV 다운로드
- 동일 시설·장비 시간 중복 방지


## Responsive update

- 모바일에서 예약 현황 표를 카드형 목록으로 전환했습니다.
- 모바일에서 월간 캘린더 대신 일자별 예약 목록을 표시합니다.
- 데스크톱에서 헤더 버튼, 대시보드 카드, 시설 카드, 테이블 너비 정렬을 보정했습니다.
- 중간 해상도에서 시설 카드가 과도하게 좁아지지 않도록 5열에서 3열로 전환됩니다.

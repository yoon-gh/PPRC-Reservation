# 🌱 표현체 연구시설 예약 시스템

본 시스템은 **표현체 연구시설의 재배 및 촬영 장비 예약을 효율적으로 관리하기 위한 웹 기반 플랫폼**입니다.
재배시설과 촬영시설을 구분하여 운영하고, 예약 승인 및 일정 관리를 체계적으로 지원합니다.

---

## 🔗 서비스 바로가기

👉 https://nihhs-prc-reservation.vercel.app/

---

## 🧩 주요 기능

### 📌 사용자 기능

* 재배시설 / 촬영시설 예약 신청
* 예약 중복 자동 검증
* 월간 캘린더 기반 일정 확인
* 예약 현황 조회

### 🛠 관리자 기능

* 예약 승인 / 반려 / 수정 / 삭제
* 점검 일정 등록 및 관리
* 예약 데이터 CSV 다운로드
* 관리자 로그인 기반 권한 제어

---

## 🧪 시스템 구성

* 재배시설: 컨베이어 온실, XYZ 생장실, 인공환경재배실 등
* 촬영시설: 다중영상촬영실, 초분광/다분광/열화상 장비 등

👉 재배(장기) / 촬영(시간 단위) 예약을 분리하여 관리

---

## ⚙️ 사용 방법

### 1. 예약 신청

* 사용자 페이지 → "예약 신청" 입력
* 시설 및 일정 선택 후 저장

### 2. 관리자 승인

* 관리자 로그인 후 예약 승인 또는 반려

### 3. 일정 확인

* 캘린더 및 예약 현황에서 확인

---

## 🔒 예약 규칙

* 동일 시설·장비의 시간 중복 예약은 불가
* 점검 일정도 예약과 동일하게 중복 방지 적용
* 승인된 예약만 최종 확정
* 미사용 또는 무단 예약 시 제한 가능
